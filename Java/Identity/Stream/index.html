# Stream流

# 概述

Java8 的 Stream 使用的是函数式编程的模式，它可以被用来对集合或数组进行链状流式的操作，可以更方便的让我们对集合或数组操作

# 快速入门

需求：

​ 打印所有年龄小于18的作家的名字，并且要注意去重

public static void main(String[] args) {
        List<Author> authors = getAuthor();
        authors
                .stream()       // 将集合转换成流
                .distinct()     // 去重
                .filter(author -> author.getAge() < 18) //过滤
                .forEach(author -> System.out.println(author.getName()));   // 遍历

    }

# 常用操作

# 创建流

数组：Arrays.stream(数组) 或者 Stream.of(数组) 来创建

Integer[] arr = {1, 2, 3, 4, 5};
Stream<Integer> stream = Arrays.stream(arr);
Stream<Integer> stream1 = Stream.of(arr);

单列集合[如List]：集合对象.stream()

List<Author> authors = getAuthors();
Stream<Author> authorStream = authors.stream();

双列集合[如Map]：转换成单列集合后再创建

HashMap<String, Integer> map = new HashMap<>();
Stream<Map.Entry<String, Integer>> entryStream = map.entrySet().stream();

# 中间操作

# filter - 过滤

Stream<T> filter(Predicate<? super T> predicate);

可以对流中的元素进行条件过滤，符合过滤条件的才能继续留着流中

例如：

​ 打印所有年龄小于18的作家的名字

public static void main(String[] args) {
	List<Author> authors = getAuthors();
	authors
       .stream()
       .filter(author -> author.getAge() < 18) 
       .forEach(author -> System.out.println(author.getName()));
}

# map

可以对流中的元素进行计算或者其它数据转换

<R> Stream<R> map(Function<? super T, ? extends R> mapper);

例如：

​ 转换：打印所有作家的姓名

public static void main(String[] args) {
        List<Author> authors = getAuthors();
        authors.stream()
                .map(author -> author.getName())
                .forEach(s -> System.out.println(s));

    }

​ 计算：将数值转为平方打印

private static void testMap(){
    Integer[] arr = {1, 2, 3, 4, 5};
    Arrays.stream(arr)
            .filter(integer -> integer > 0)
            .map(integer -> integer * integer)
            .forEach(integer -> System.out.println(integer));
}

# mapToInt - 基本数据类型的优化

可以减少大量的拆箱和装箱

private static void testMap(){
    Integer[] arr = {1, 2, 3, 4, 5};
    Arrays.stream(arr)
            .filter(integer -> integer > 0)
            .map(integer -> integer * integer)
            .forEach(integer -> System.out.println(integer));
    Arrays.stream(arr)
            .mapToInt(integer -> integer * integer)
            .forEach(System.out::println);
}

# distinct - 去重

可以去除流中的重复元素

注意：distinct方法是依赖Object的equals方法来判断是否是相同对象的，所以元素对应对象要重写equals()和hashCode()

例如：

​ 打印去重后的作家名字

private static void testDistinct(){
    List<Author> authors = getAuthors();
    authors.stream()
            .distinct()
            .forEach(author -> System.out.println(author.getName()));
}

# sorted - 排序

可以对流中的元素进行排序

注意，如果使用空参方法，元素对应的类需要实现

例如：

​ 对流中的元素按照年龄进行降序排序，并且要求不能有重复元素

@Data
@AllArgsConstructor
@NoArgsConstructor
@EqualsAndHashCode
public class Author implements Comparable<Author> {
    private Long id;
    private String name;
    private Integer age;
    private String intro;
    private List<Book> books;

    @Override
    public int compareTo(Author author) {
        return author.getAge() - this.getAge();
    }
}

private static void testSorted(){
        List<Author> authors = getAuthors();
        authors.stream()
                .distinct()
                .sorted(new Comparator<Author>() {
                    @Override
                    public int compare(Author o1, Author o2) {
                        return o2.getAge() - o1.getAge();
                    }
                })
                .forEach(new Consumer<Author>() {
                    @Override
                    public void accept(Author author) {
                        System.out.println(author);
                    }
                });
    }

# limit

可以设置流的最大长度，超出的部分将被抛弃

例如：

​ 对流中的元素按照年龄进行降序排序，并且要求不能有重复的元素，然后打印其中年龄最大的两个作家的姓名

private static void testLimit(){
    List<Author> authors = getAuthors();
    authors.stream()
            .distinct()
            .sorted((o1, o2) -> o2.getAge() - o1.getAge())
            .limit(2)
            .forEach(author -> System.out.println(author.getName()));
}

# skip

跳过流中的前n个元素，返回剩下的元素

例如：

​ 打印除了年龄最大的作家外的其他作家，要求不能有重复元素，并且按照年龄排序降序

private static void testSkip(){
    List<Author> authors = getAuthors();
    authors.stream()
            .distinct()
            .sorted()
            .skip(1)
            .forEach(author -> System.out.println(author.getName()));
}

# flatMap

map只能把一个对象转换成另一个对象来作为流中的元素。而flatMap可以把一个对象转换成多个对象作为流中的元素

例如：

​ 打印所有书籍的名字，要求对重复的元素进行去重

private static void testFlatMap(){
    List<Author> authors = getAuthors();
    authors.stream()
            .flatMap(new Function<Author, Stream<Book>>() {
                @Override
                public Stream<Book> apply(Author author) {
                    return author.getBooks().stream();
                }
            })
            .distinct()
            .forEach(new Consumer<Book>() {
                @Override
                public void accept(Book book) {
                    System.out.println(book.getName());
                }
            });
}

例如：

​ 打印现有书籍的所有分类，要求对分类进行去重，不能出现这种格式：哲学，爱情

private static void testFlatMap2(){
    List<Author> authors = getAuthors();
    authors.stream()
            .flatMap(new Function<Author, Stream<Book>>() {
                @Override
                public Stream<Book> apply(Author author) {

                    return author.getBooks().stream();
                }
            })
            .distinct()
            .flatMap(new Function<Book, Stream<String>>() {
                @Override
                public Stream<String> apply(Book book) {
                    return Arrays.stream(book.getCategory().split(",").clone());
                }
            })
            .distinct()
            .filter(new Predicate<String>() {
                @Override
                public boolean test(String s) {
                    return !"哲学".equals(s) && !"爱情".equals(s);
                }
            })
            .forEach(new Consumer<String>() {
                @Override
                public void accept(String s) {
                    System.out.println(s);
                }
            });
}

# count

可以用来获取当前流中元素的个数

例子：

​ 打印这些作家的所出书籍的数目，注意删除重复元素

# max&min

可以用来或者流中的最值

例子：

​ 分别获取这些作家的所出书籍的最高分和最低分并打印

private static void testMaxAndMin(){
    List<Author> authors = getAuthors();
    Optional<Book> max = authors.stream()
            .flatMap(new Function<Author, Stream<Book>>() {
                @Override
                public Stream<Book> apply(Author author) {
                    return author.getBooks().stream();
                }
            })
            .distinct()
            .max(new Comparator<Book>() {
                @Override
                public int compare(Book o1, Book o2) {
                    return o1.getScore() - o2.getScore();
                }
            });
    System.out.println("max：" + max.get().getScore());

}

# collect

把当前流转换成一个集合

提供了 Collectors 工具类

例子：

​ 获取一个存放所有作者名字的List集合

private static void testCollect(){
    List<Author> authors = getAuthors();
    List<String> collect = authors.stream()
            .distinct()
            .map(new Function<Author, String>() {
                @Override
                public String apply(Author author) {
                    return author.getName();
                }
            })
            .collect(Collectors.toList());
    System.out.println(collect);
}

​ 获取一个所有书名的Set集合

private static void testCollect2(){
    // 获取一个所有书名的Set集合
    List<Author> authors = getAuthors();
    Set<String> collect = authors.stream()
            .flatMap(new Function<Author, Stream<Book>>() {
                @Override
                public Stream<Book> apply(Author author) {
                    return author.getBooks().stream();
                }
            })
            .map(new Function<Book, String>() {
                @Override
                public String apply(Book book) {
                    return book.getName();
                }
            })
            .collect(Collectors.toSet());
    System.out.println(collect);
}

​ 获取一个Map集合，key为作者名，value为List

private static void testCollect3(){
    // 获取一个Map集合，key为作者名，value为List<Book>
    List<Author> authors = getAuthors();
    Map<String, List<Book>> collect = authors.stream()
        	.distinct()
            .collect(Collectors.toMap(author -> author.getName(), author -> author.getBooks()));
}

# reduce - 归并

对流中的数据按照你指定的计算方式计算出一个结果

reduce的作用是把stream中的元素给组合起来，我们可以传入一个初始值，它会按照我们的计算方式一次拿流中的元素和初始化值进行计算，计算结果再和后面的元素计算

1] T reduce(T identity, BinaryOperator<T> accumulator) 内部计算方式：

T result = identity;
for (T element : this stream) {
	result = accumulator.apply(result, element);
}
return result;

​ 其中identity就是我们可以通过方法参数传入的初始值，accumulator的apply具体进行什么计算也是我们通过方法参数来确定的

例子：

​ 使用 reduce 求所有作者年龄的和

private static void testReduce(){
//        使用 reduce 求所有作者年龄的和
        List<Author> authors = getAuthors();
        Integer reduce = authors.stream()
                .distinct()
                .map(new Function<Author, Integer>() {
                    @Override
                    public Integer apply(Author author) {
                        return author.getAge();
                    }
                })
                .reduce(0, new BinaryOperator<Integer>() {
                    @Override
                    public Integer apply(Integer result, Integer element) {
                        return result + element;
                    }
                });
        System.out.println(reduce);
    }

2] Optional<T> reduce(BinaryOperator<T> accumulator) 内部计算方式

boolean foundAny = false;
     T result = null;
     for (T element : this stream) {
         if (!foundAny) {
             foundAny = true;
             result = element;
         }
         else
             result = accumulator.apply(result, element);
     }
     return foundAny ? Optional.of(result) : Optional.empty();

​ 将取出的第一个元素作为初始值

例：

​ 使用reduce求所有作者中年龄最大值

private static void testReduce2(){
//        使用reduce求所有作者中年龄最大值
        Optional<Integer> reduce = getAuthors()
                .stream()
                .distinct()
                .map(new Function<Author, Integer>() {

                    @Override
                    public Integer apply(Author author) {
                        return author.getAge();
                    }
                })
                .reduce(new BinaryOperator<Integer>() {
                    @Override
                    public Integer apply(Integer integer, Integer integer2) {
                        return integer < integer2 ? integer2 : integer;
                    }
                });
        System.out.println(reduce.get());
    }

# 并行流

当流中有大量元素时，可以使用并行流去提高操作的效率。其实并行流就是把任务分配给多个线程去完成。如果使用Stream，只需修改一个方法的调用就可以使用并行流实现，从而提高效率

parallel()

// 测试并行流
private static void testParallel() {
    Stream<Integer> stream = Stream.of(1, 2, 3, 4, 5, 6, 7, 8);
    Integer integer = stream.parallel()
            .filter(new Predicate<Integer>() {
                @Override
                public boolean test(Integer integer) {
                    return integer > 4;
                }
            })
            .reduce(new BinaryOperator<Integer>() {
                @Override
                public Integer apply(Integer integer, Integer integer2) {
                    return integer + integer2;
                }
            })
            .get();
    System.out.println(integer);
}

也可以直接使用 parallelStream() 获取并行流

List<Author> authors = getAuthors();
Stream<Author> authorStream = getAuthors().parallelStream();

# 注意事项

  • 惰性求值 (如果没有终结操作，没有中间操作时不会得到执行的)
  • 流是一次性的 (一旦一个流对象经过一个终结操作后。这个流就不能再被使用)
  • 不会影响原数据 (我们在流中可以对数据做很多处理。但是正常情况下是不会影响原来集合中的元素的)
