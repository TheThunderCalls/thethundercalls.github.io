# Lambda表达式

# 概念

Lambda 表达式是 JDK8 中的一个语法糖。它可以对某些匿名内部类的写法进行简化。是函数式编程思想的一个重要体现。让我们不用关注是什么对象，而是更关注对数据进行了什么操作

# 入门

# 基本格式

(参数列表) -> {代码块}

# 省略规则

  • 参数类型可以省略
  • 方法体中只有一句代码时， {}、return 和 ; 都可以省略
  • 方法只有一个参数时， () 可以省略

# 示例

# 函数式接口

只有一个抽象方法的接口我们称之为函数式接口

JDK 的函数式接口都加上了 @FunctionalInterface 注解进行标识，但是无论是否加上该注解，只要接口中只有一个抽象方法，都是函数式接口

# 特性

是一个有且仅有一个抽象方法，但是可以有多个非抽象方法的接口

在一个接口上使用 @FunctionalInterface 注解，可以检查它是否是一个函数式接口

只要一个对象是函数式接口的实例，那么该对象就可以用 Lambda 表达式来表示

# 内置核心函数式接口

函数式接口 参数类型 返回值类型 用途
Consumer T void 消费型接口，对类型为 T 的对象应用操作
Supplier 无 T 供给型接口，返回类型为 T 的对象
Function<T, R> T R 函数型接口，对类型为 T 的对象应用操作，并返回 R 类型的对象的结果
Predicate T boolean 断定型接口，确定类型为 T 的对象是否满足某约束，返回 boolean 值
